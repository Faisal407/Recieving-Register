Auto-create Google Sheet / OneDrive

This is not something a static offline page can do safely, and here's exactly why:

Creating a file directly in your Google Drive or OneDrive requires the page to sign in as you via Google/Microsoft's official login (OAuth) and hold a server-side app registration — that's account infrastructure, not something to embed in a client-side test page. Doing it insecurely (e.g. hardcoding access) would risk your account credentials.

What this prototype does instead: Export Excel downloads a real .xlsx you can upload to Drive/OneDrive manually in a few seconds.
What the real app will do: once we build the Flutter app, it'll use a proper Google Sheets API service account (or Microsoft Graph for OneDrive) to push each record automatically — that's a backend feature for the next build phase, not this test page.
