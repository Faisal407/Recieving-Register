☁ Cloud Sync — Apps Script Setup

One-time setup, reusable for every project. Uses a free Google Apps Script Web App — no OAuth login needed in this page, since the script runs under your own Google account's authorization once deployed. Each project links its own Sheet ID + Web App URL from its card in the Projects tab.

1. Open (or create) the Google Sheet for a project.
2. Extensions → Apps Script.
3. Delete any starter code, paste the script below.
4. Replace PASTE_YOUR_SHEET_ID_HERE with that Sheet's ID (from its URL, between /d/ and /edit).
5. Deploy → New deployment → Type: Web app → Execute as: Me → Who has access: Anyone.
6. Copy the URL (ends in /exec) into that project's card on the Projects tab.

Known limitation: the browser can't read the script's response (Apps Script doesn't send CORS headers), so this page can't confirm each row landed — it marks "Sent (unconfirmed)". Keep exporting Excel per project as your reliable backup. OneDrive auto-export has no equivalent no-login trick — that genuinely needs the Phase-2 backend with Microsoft sign-in.
